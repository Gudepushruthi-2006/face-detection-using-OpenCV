# Face Detection using OpenCV

## Project Description

This project detects human faces in an image using Python and OpenCV. The program uses a Haar Cascade Classifier to identify faces and draws rectangles around the detected faces.

## Technologies Used

* Python
* OpenCV
* NumPy
* Google Colab

## How the Project Works

1. An image is uploaded to the system.
2. The image is converted into grayscale.
3. The Haar Cascade face detection model scans the image.
4. If a face is detected, a rectangle is drawn around it.
5. If no face is detected, the output shows "No face detected".

## Example Output

* Face detected: 1
* No face detected

## Applications

* Security systems
* Attendance systems
* Camera face detection
